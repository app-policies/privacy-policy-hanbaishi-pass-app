# 販売士Pass プライバシーポリシー

制定日: 2026-06-14

販売士Pass（以下「本アプリ」）は、学習機能、広告配信、課金機能、利用状況の改善のために必要な範囲で情報を取り扱います。

## 取得する情報

本アプリは、以下の情報を取得または利用する場合があります。

- アプリ内の学習履歴、正誤履歴、設定情報
- 広告配信および広告視聴による問題追加解放に必要な端末情報、広告識別子、広告表示に関する情報
- 課金状態を確認するための購入情報
- アプリの品質改善に必要な利用状況に関する情報

## 利用目的

取得した情報は、以下の目的で利用します。

- 学習履歴、誤答復習、進捗表示などの機能提供
- 広告配信と広告表示の最適化
- プレミアム購入状態の確認
- 不具合調査、品質改善、利用状況の分析
- お問い合わせへの対応

## 第三者サービス

本アプリでは、以下の第三者サービスを利用する場合があります。

- Google Mobile Ads
- RevenueCat
- Firebase Analytics
- Apple

各サービスにより取得される情報や利用目的は、各サービス提供者のプライバシーポリシーをご確認ください。

Firebaseは利用状況の分析目的に限って利用します。認証、データベース、クラウド機能、ストレージ、リモート設定、クラッシュ収集のためには利用しません。
